磨皮

磨皮操作可实现两种效果：提升人物质感，对光影的调整。

Portraiture 插件

提升人物质感。使用使首先用吸管工具吸取一下中间调区域。

DR5 插件

调整光影。

高反差磨皮

  1. 复制图层，设置图层混合模式为线性光。（相当于增强了对比度）
  2. command + i 反向。
  3. 滤镜-其他-高反差保留，设置半径。
  4. 滤镜-模糊-高斯模糊，半径=高反差半斤/2～高反差。
  5. 添加黑色图层蒙版，用白色画笔涂抹需要磨皮区域。

    中性灰磨皮

  6. 黑白调整图层 去色观察图层（去除颜色对视觉的干扰）
  7. 曲线调整图层 明暗观察图层（command + g可以将图层分组）
  8. command + shift + n 弹出新建图层界面，选择柔光混合模式,勾选填充50%中性灰选项。
  9. 中性灰编辑图层，比50%灰亮的地方提亮基色，比50%暗的地方压暗基色。
  10. 画笔硬度0，不透明度3%-10%。画笔颜色：白色提亮，黑色压暗。D键矫正画笔颜色为黑白色。X键切换画笔颜色。

形体形态调整

立七坐五盘三半

腿部拉长

  1. 矩形选框选择大腿/小腿及以下部分。
  2. command+j 复制选中图层。
  3. command+t 自由变换。
  4. shift + 鼠标拖动，拉长腿部。
  5. command+option+shift+e 盖印一个图层，便于后续处理。

液化工具

常用工具有向前变形工具，褶皱工具，膨胀工具、冻结蒙版、解冻蒙板工具。使用技巧：

  1. 根据不同位置调整画笔大小。
  2. 画笔压力建议在10-20，不易过大。
  3. 褶皱工具和膨胀工具会根据画笔速率导致影像做向内向外的畸变。
  4. 冻结蒙板可以使涂抹的位置固定，从而不发生形变。

色彩调整工具

色彩基础知识

  • 饱和度（saturation）：提升所有颜色的强度，可能导致过饱和，局部细节消失，最常见是皮肤的过饱和（变成橙色且不自然）。
  • 自然饱和度：智能提升画面中比较柔和（即饱和度低）的颜色，而使原本饱和度够的颜色保持原状。类似于对照片补光，但是是对颜色的补光。可以防止皮肤颜色变得过饱和以及不自然。
  • 色相饱和度，调整图像中特定颜色范围的色相、饱和度和明度，或者同时调整图像中的所有颜色。此调整尤其适用于微调 CMYK 图像中的颜色，以便它们处在输出设备的色域内。

自然饱和度

快捷键： 图像-调整-自然饱和度

色相饱和度

快捷键： command+u

色彩平衡

快捷键： command+b
